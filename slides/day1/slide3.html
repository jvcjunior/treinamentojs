Exemplo 1



    var foo= "bar";

    function bar(){ 
        var foo = "baz";
    }

    function baz(){
        foo = "bam";
        bam = "hello";
    }
    

    
Compilação Javascript

    
Linha 1: Ei escopo global, eu tenho a declaração de uma variável chamada foo;

    
Linha 2: Ei escopo global, eu tenho a declaração de uma função* chamada bar;

    
Linha 4: Ei escopo de bar, eu tenho a declaração de uma variável chamada foo;

    
Linha 7: Ei escopo global, eu tenho a declaração de uma função chamada baz;

    
Linha 7.5: Ei escopo de baz, eu tenho a declaração de uma variável chamada foo;
