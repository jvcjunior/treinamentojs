Exemplo 1



    var foo= "bar";

    function bar(){ 
        var foo = "baz";
    }

    function baz(){
        foo = "bam";
        bam = "hello";
    }
    

    
Execução Javascript

    
    

        Linha 1: Ei escopo global, eu tenho uma declaração LHS de uma variável chamada foo; Conhece?
    

    

        O escopo global conhece porque foo foi declarada na linha 1 da fase de compilação, então a atribuição ocorre. 
    

    
    

        Linha 4: Ei escopo de bar, eu tenho a declaração LHS de uma variável chamada foo; Conhece?
    

    

        O escopo de bar conhece porque foo foi declarada na linha 4 da fase de compilação, então a atribuição ocorre. 
    

    
    

        Linha 8: Ei escopo de baz, eu tenho a declaração LHS de uma variável chamada foo; Conhece?
    

    

        O escopo de baz conhece porque foo foi declarada na linha 7.5 da fase de compilação, então a atribuição ocorre. 
    

    
    

        Linha 9: Ei escopo de baz, eu tenho a declaração LHS de uma variável chamada bam; Conhece?
    

    

        Linha 9: Ei escopo global, eu tenho a declaração LHS de uma variável chamada bam; Conhece?
    

    

       O escopo global conhece porque bam criada*, então a atribuição ocorre.
