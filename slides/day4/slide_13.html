Closure: classic module pattern


var foo = (function(){
    var o = { bar : "bar"};
    
    return { 
        bar : function(){
            console.log(o.bar);
        }
    };
})();

foo.bar();  // "bar"
Classic module pattern. 2 Características: 1. Tem de haver uma função externa executada. 2. Tem de haver uma ou mais funções que são retornadas daquela chamada de função que tem um closure sobre o escopo privado interno.
